# Personal AI Hub

> Personal AI Operations Hub
> AI Agent 規範中心 × Prompt 知識庫 × 開發實驗室 × 個人戰略中樞

## Purpose

這個 Repo 不是一般的程式碼倉庫，而是你的 **AI Operating System (AIOS) 知識中樞**。

集中存放:

- AI Agent 規範（AGENTS / CLAUDE / SOUL / PRINCIPLES）
- Prompt 模板庫（system / workflow / coding / research / management）
- AI 工具使用紀錄（Claude / Copilot / Gemini / Cursor / Jules / Other）
- 開發小專案與實驗（projects / experiments）
- 知識庫（architecture / ai-agents / productivity / strategy）
- 工作流範本（planning / learning / coding / decision-making）
- 個人記憶與決策紀錄

整體定位:

```
AI Operating System   ← 母系統（本 Repo）
├── Idea Discussion   ← 子專案
├── Automation
└── Future Projects
```
